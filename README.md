## 🚀 Why UI‑Playground?

- **⚡ 500+ production‑ready components** – buttons, cards, loaders, forms, feedback, navigation, data display, effects & layouts.
- **🎨 Premium dark theme** – gradient borders, glassmorphism, neon glows, smooth animations.
- **📋 One‑click copy** – HTML, CSS & JS extracted directly from the live demo.
- **🔍 Smart filtering** – search by name, category, or tag (CSS/JS).
- **📱 Fully responsive** – works on all devices, sidebar collapses on mobile.
- **🔓 No login, completely free** – open source under MIT license.
